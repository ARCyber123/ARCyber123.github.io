How to Host Your Website On GitHub

GitHub is a great software development platform. Here, I will show you how to host a website with GitHub Pages, which is an easy way to deploy a website from a GitHub repository.

1. Sign Up for GitHub

If you already have a GitHub account, skip to Step 2.
Click here to sign up for GitHub.

You will see a sign-up form. Enter a username, email address and password. You can choose below if you want to receive product updates, announcements and offers through email. To verify your account, scroll down and click "Verify" and solve the puzzle.

Once you have filled everything in, click "Create account". Next, you will have to fill in some more details to personalize GitHub. Then, you will be asked to verify your email address. Go to your inbox and open the verification email. Click the verification link to verify your email address. Once all that is done, your account is ready to go.

2. Creating a Repository

To create a new repository, click "New" on the left.

Now, make sure you name the repository exactly "YourUsername.github.io" and replace "YourUsername" with your username. Naming it right is important. If you name it wrong, your website won't be published!

After you're done, click "Create repository".

3. Upload/Create Files

Now once your repository is created, it's time to fill it up! if you already have the files for your website on your computer, click "Upload files". There, you can drag and drop files or open the file chooser. If you haven't created any files for the website, you can just click "Create new file". Name it "index.html", because that's the name web browsers expect for the main page of your website. Type your code, click "Commit changes" and your website will be published soon. To view your website, type "YourUsername.github.io" (replace "YourUsername" with your actual username) and use the shortcut "Ctrl + Shift + R" or "Cmd + Shift + R" until your website appears. Why that shortcut? Because you'll need to clear the browser cache and reload the site to see it quickly. Cache files for a website are stored by the browser to speed up the loading time of a website.

4. Celebrate!

Now that your website is ready, share it with your family, friends and celebrate!
