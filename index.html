Basic Info About HTML

HTML (HyperText Markup Language) is the programming language used to create web pages on the Internet. It was invented by Tim Berners-Lee for the Internet. CSS is another important language which goes along with HTML. It is used just for styling and customisation of your website. This website was made with just HTML & CSS! Basic knowledge of HTML & CSS is absolutely required if you're making a website. Learn more about the birth of the internet at CERN.

Tim Berners-Lee

A Few HTML Tags

a = link
p = paragraph
h1, h2, h3, h4, h5, h6 = headings
img = image
strong = bold text
em = emphasis (italicise text)
br = line break (next line)

Highly Used CSS (there's a lot more!)

font-family = font family
font-weight = font thickness
font-style = font style (italic, cursive etc.)
font-size = font size
background-color = background colour
color = element colour
